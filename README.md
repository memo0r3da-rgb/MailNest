# MailNest Temporary Email

موقع بريد مؤقت عربي جاهز للنشر كنسخة static، مع صفحات محتوى وثقة ولوحة تحكم محلية.

## التشغيل المحلي

```powershell
python -m http.server 4173
```

ثم افتح:

```text
http://127.0.0.1:4173
```

## النشر المجاني المقترح

أفضل بداية مجانية:

1. Cloudflare Pages
   - مجاني وسريع.
   - يعطيك رابط مجاني مثل `mailnest.pages.dev`.
   - يدعم ملفات `_headers` و`_redirects` الموجودة في المشروع.

2. Netlify
   - مجاني وسهل بالسحب والإفلات.
   - يعطيك رابط مجاني مثل `mailnest.netlify.app`.
   - يدعم `_headers` و`_redirects`.

3. Vercel
   - مجاني ومناسب للواجهات.
   - يستخدم `vercel.json` الموجود في المشروع.

لـ Google AdSense: استخدم الاستضافة المجانية للتجربة والفهرسة الأولية، لكن الأفضل شراء دومين حقيقي لاحقا وربطه بالموقع قبل التقديم. روابط subdomain المجانية قد تعمل للعرض، لكنها غالبا ليست الخيار الأقوى لقبول AdSense.

بعد ربط الدومين الحقيقي:

1. غيّر `https://example.com` في `sitemap.xml`.
2. غيّر canonical داخل `index.html`.
3. غيّر `pub-0000000000000000` داخل `ads.txt` بعد إنشاء حساب AdSense.
4. أرسل الموقع إلى Google Search Console.
5. قدّم على AdSense بعد إضافة محتوى كاف وزيارات أولية.

## الملفات المهمة

- `index.html`: الصفحة الرئيسية وصندوق البريد المؤقت.
- `premium.html`: صفحة الخطة المدفوعة وطرق الدفع المحلية داخل مصر.
- `blog.html` و`article.html`: المقالات.
- `about.html`, `privacy.html`, `terms.html`, `faq.html`, `contact.html`: صفحات ثقة مهمة للمستخدمين وAdSense.
- `admin.html`: لوحة التحكم المخفية، غير مرتبطة من الموقع.
- `robots.txt`, `sitemap.xml`, `ads.txt`: ملفات نشر وفهرسة وإعلانات.
- `assets/mailnest-logo.png`: أصل اللوجو المستخدم في الهيدر والفوتر وبيانات المشاركة.

## لوحة التحكم

افتح:

```text
http://127.0.0.1:4173/admin.html
```

بيانات الدخول التجريبية:

```text
admin@mailnest.local
MailNest@2026
```

لوحة التحكم الحالية تحفظ المحتوى في `localStorage` داخل نفس المتصفح. هذا مناسب لمرحلة static demo، لكن للنشر التجاري الحقيقي يفضل لاحقا إضافة باك إند وقاعدة بيانات وتسجيل دخول آمن.

## AdSense

تمت إضافة صفحات الخصوصية والشروط والتواصل والمقالات والتنقل الواضح ومساحات الإعلانات. قبل التقديم:

1. استبدل `example.com` في `sitemap.xml` ووسوم canonical بدومينك الحقيقي.
2. استبدل `pub-0000000000000000` في `ads.txt` برقم الناشر من Google AdSense.
3. أضف كود AdSense الحقيقي داخل عناصر `.ad-banner` و`.ad-slot`.
4. أضف مقالات أصلية أكثر قبل التقديم، وراجع سياسات Google AdSense الرسمية.

لا يوجد ضمان قبول تلقائي، لأن Google تراجع جودة المحتوى والالتزام بالسياسات وتجربة التنقل.
